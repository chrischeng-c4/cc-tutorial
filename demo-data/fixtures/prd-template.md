# PRD Template

## 1. Problem
- What user problem are we solving?
- Why does it matter now?

## 2. Goals
- User outcome
- Business outcome
- Success metrics

## 3. Non-goals
- What is explicitly out of scope?

## 4. User Stories
- As a user, I want ...

## 5. Requirements
- Functional requirements
- Permission / security requirements
- Data and reporting requirements

## 6. Edge Cases
- Empty state
- Large data
- Permission denied
- Partial failure

## 7. Open Questions
- Product questions
- Engineering questions
- Legal / compliance questions

## 8. Launch / Rollout
- Experiment or staged rollout
- Monitoring
- Rollback plan
